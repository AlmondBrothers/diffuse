## Usage
This app will create silent disco playlists accessible by the location of the user.

## Requirements

- Node 0.10.x
- Express
- Postgresql 9.1.x
- React


### Installing Dependencies and Running the Server

From within the root directory:

```sh
npm install
npm start
npm run webpack
```
